CSS Shadows Guide

Learn how to create beautiful shadows in HTML/CSS

1. Box Shadow

The box-shadow property adds shadow effects around an element's frame.

box-shadow: [horizontal] [vertical] [blur] [spread] [color] [inset];

Simple Shadow

box-shadow: 0 4px 8px rgba(0,0,0,0.1);

Blurred Shadow

box-shadow: 0 10px 20px rgba(0,0,0,0.19);

Inset Shadow

box-shadow: inset 0 0 10px rgba(0,0,0,0.3);

Multiple Shadows

box-shadow: 0 1px 3px, 0 1px 2px;

2. Text Shadow

The text-shadow property adds shadows to text.

text-shadow: [horizontal] [vertical] [blur] [color];
Text Shadow

text-shadow: 2px 2px 4px rgba(0,0,0,0.3);

Glow Effect

text-shadow: 0 0 10px #3498db;

3. Filter: Drop Shadow

The filter: drop-shadow() applies a shadow to an element, including non-rectangular shapes.

filter: drop-shadow([horizontal] [vertical] [blur] [color]);

filter: drop-shadow(8px 8px 10px rgba(0,0,0,0.5));

4. Neumorphism

A modern design trend that uses shadows to create soft, extruded shapes.

Neumorphic Design

Combining light and dark shadows

Shadow Properties Explained

Horizontal Offset
Positive values move shadow right, negative values move it left.
Vertical Offset
Positive values move shadow down, negative values move it up.
Blur Radius
Higher values create more blur (0 = sharp shadow).
Spread Radius
Positive values expand the shadow, negative values shrink it.
Color
Shadow color (RGBA recommended for transparency).
Inset
Changes the shadow from outer to inner shadow.
